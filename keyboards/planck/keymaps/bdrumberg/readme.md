# The Default Planck Layout

## QMK Cheatsheet

- [Tutorial](https://docs.qmk.fm/#/newbs)
- `qmk compile`
- Press reset on bottom
- `qmk flash`

## Notes

- Switching layers
  - MO(layer) - momentarily activate a layer
  - TG(layer) - toggle layer on and off
